Creative Coding Projects (p5.js)

Digital DJ Board

A sound manipulation-focused project where I collaborated with a partner to create a digital DJ board that allows users to remix songs from different regions of the world. Three songs from each continent (excluding Australia) have been preloaded into the sketch. The user has the option to change the speed and distortion levels of the song, and create an echo effect. There are also some fun sound effects. Users can record and download their remixed song, and save it as a .wav file on their desktops.

24HR Clock

A time-based media project where I have recreated the Apple Music now playing view, except instead of a 3-4 minute song, The current “song” is a 24-hour day. The song title is displayed as the current date, the artist is the current year, and the progress line inches on as the day goes. In the AM, the color pallet for the song cover turns darker and gets brighter in the AM.
